# Projet BAC PRO CIEL — Maquettes Électroniques

Bienvenue dans le dépôt de présentation du **BAC PRO CIEL** (Cybersécurité, Informatique et réseaux, Électronique).

Ce projet regroupe **4 maquettes électroniques** réalisées dans le cadre de la formation. Pour chaque maquette, vous trouverez le code source ainsi que les schémas électriques associés.

---


## Comment contribuer

1. Déposez vos fichiers de code dans le sous-dossier `code/` de la maquette concernée.
2. Déposez vos schémas électriques (PDF, PNG, KiCad, etc.) dans le sous-dossier `schemas/`.
3. Mettez à jour le `README.md` de la maquette avec une description, la liste des composants et les instructions de montage.
