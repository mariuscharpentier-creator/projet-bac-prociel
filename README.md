# 📚 Projet BAC — Prociel

> Projet de terminale développé dans le cadre du baccalauréat, en lien avec la spécialité **NSI** (Numérique et Sciences Informatiques) ou similaire, réalisé sous la supervision de l'établissement **Prociel**.

---

## 📝 Description

Ce dépôt contient l'ensemble des ressources, du code source et de la documentation associés au projet de BAC **Prociel**.  
Il regroupe toutes les étapes du développement : conception, implémentation et présentation finale.

## 🚀 Installation & Utilisation

1. **Cloner le dépôt**
   ```bash
   git clone https://github.com/mariuscharpentier-creator/projet-bac-prociel.git
   cd projet-bac-prociel
   ```

2. **Consulter la documentation**  
   Ouvrez ce fichier `README.md` ou parcourez les dossiers du projet.
